feign的基本用法

发表于2023-05-25|JAVARPCFEIGN
@FeignClient(value = "bds-server", url = "${remote.bds-url}")
public interface BasicDataFeign {

    //http://fns-yapi.sit.sf-express.com/project/17/interface/api/7236
    @PostMapping("/site/pageSiteByCity")
    Result<Page<RemoteSiteCodeResponse>> getSiteList(RemoteSiteRequest request);

    @GetMapping("/site/siteById")
    Result<RemoteSiteResponse> siteById(@RequestParam("siteId") Long siteId);

}

GET请求传单个参数

发送GET请求的时候，直接使用是不对的。比如下面这样，就是错误的

@GetMapping("/site/siteById")
Result<RemoteSiteResponse> siteById(Long siteId);

必须要加上@RequestParam("siteId") 注解。才可以。

加上注解之后，get请求的参数才会加上。否则是没有后面的参数的。

http://10.206.173.207/bds-server/site/siteById?siteId=57383

GET请求传多个参数

那如果要传多个参数呢？

方法一

@FeignClient(name = "demoFeignClient")
public interface DemoFeignClient {
    @GetMapping("/test")
    public User test(@SpringQueryMap User user);
}

方法二

把参数平铺出来，每个参数加上@RequestParam注解。

@FeignClient(name = "demoFeignClient")
public interface DemoFeignClient {
  @GetMapping(value = "/test")
  public User test(@RequestParam("id") Long id, @RequestParam("name") String name);
}

方法三

使用Map的方式

@FeignClient(name = "demoFeignClient")
public interface DemoFeignClient {
  @GetMapping(value = "/test")
  public User test(@RequestParam Map<String, Object> map);
}
