SourceTree提交报错Authentication failed for

原因是因为系统中存的用户名和密码不对。

最简单的操作办法就是把系统中存在的用户名和密码删掉，然后重新填。

打开选项

将Git保存的密码全都删掉

然后再次重新提交，系统就会提示需要重新登录了。
