git批量删除分支

批量删除除当前分支外的所有本地分支：

git branch | xargs git branch -d

批量删除指定分支，比如删除所有，分支名字包含zspc的分支

git branch | grep ‘zspc’ | xargs git branch -d

删除的过程中，如果被删除的分支没有merge，会有提示，git默认不会删除，如果我们不需要，可以使用-D强制删除
