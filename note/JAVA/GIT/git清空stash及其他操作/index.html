git清空stash及其他操作

将当前变更保存到stash

git stash

将当前变更保存到stash，自定义说明信息

git stash save ‘这里是自定义说明信息’

列出所有的stash

git stash list

弹出stash，也就是将stash中的内容应用到当前分支，并删除这个stash

git stash pop

弹出指定的stash，也就是将stash中的内容应用到当前分支，并删除这个stash，后面的@{0}可以指定

git stash pop stash@{0}

删除stash

git stash drop

删除指定的stash,后面的@{0}可以指定

git stash drop stash@{0}

清空stash

git stash clear

展示stash中的内容

git show stash@{n}

gitgit stash
