2.找到端口9090对应的PID

TCP    0.0.0.0:9090           0.0.0.0:0              LISTENING       13320

3.找到pid对应的具体进程名是哪一个

查到13320对应的是javaw.exe这个进程

C:\Users\BG402180> tasklist | findstr "13320"
javaw.exe                    13320 Console                    2    225,548 K

4.停止这个进程

C:\Users\BG402180>taskkill /f /t /im javaw.exe
成功: 已终止 PID 13320 (属于 PID 17924 子进程)的进程。
